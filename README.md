# Liquidity Sentinel

Система раннего предупреждения стресса ликвидности рублёвого денежного рынка для казначейства ПСБ.

## Возможности

| Модуль | Описание |
|--------|----------|
| **M1** | Усреднение обязательных резервов + RUONIA |
| **M2** | Аукционы репо ЦБ (7 дней), cover ratio, spread к ключевой |
| **M3** | Аукционы ОФЗ (Минфин), недоспрос / переспрос |
| **M4** | Налоговый календарь ФНС, устранение двойного счёта |
| **M5** | Казначейство (SORS, Росказна) |
| **LSI** | Агрегация 0–100, вклад модулей, backtest, sensitivity |
| **LLM** | Аналитичесткий комментарий, интерактивный чат |
